## Applications

- **Floor / lift detection** — relative altitude change from a stored baseline.
- **Weather-adjacent teaching** — slow pressure trends over minutes (not a replacement for a weather station).
- **Multi-sensor labs** — combine with IMU motion and SHT40 humidity for environmental storytelling.

## Sensor Studio workflow

1. Add **DPS368** input or tap nodes (pressure, temperature, derived altitude).
2. Wire to trend plots, numeric gauges, or threshold LEDs.
3. Align publish configuration with the Bitstream sensor settings deck.
